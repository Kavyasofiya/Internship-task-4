# Group Chat Application

A full-featured group chat application with admin controls, real-time messaging, and user management.

## Features

### Group Management
- Create groups with custom names and descriptions
- Add/remove members from groups
- Assign admin roles to members
- Toggle admin-only messaging mode
- Mute/unmute members (temporary or permanent)

### Real-time Messaging
- Instant message delivery via WebSockets
- Read receipts and typing indicators
- Support for text, images, and files
- Message deletion (soft delete)

### User Management
- User registration and authentication
- Profile management
- Online status tracking
- Search users by username or email

### Security
- JWT-based authentication
- Role-based access control
- Input validation and sanitization
- Secure password hashing
